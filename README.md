# NEXUS — Freelance & Hiring Platform

A complete, production-ready React SPA simulating a full freelance marketplace with escrow wallet.

## Quick Start

```bash
unzip nexus.zip
cd nexus
npm install
npm run dev
# → http://localhost:5173
```

## Tech Stack
- **React 18** + **React Router v6**
- **Vite 5** — dev server & build
- **CSS Modules** — scoped styles, zero deps
- **Context API + localStorage** — full state persistence across sessions
- **Google Fonts** — Cormorant (display) + DM Sans (body)

---

## Feature Map

### Auth (`/auth`)
- Login / Signup with role selection (Client or Freelancer)
- Split-panel design with animated left panel
- Form validation with error states
- Persists session to localStorage

### Home (`/`)
- Animated hero with hire/work mode toggle
- Live search with popular suggestions
- Category grid (10 categories, scroll-reveal)
- How it works section
- CTA block

### Browse Talent (`/browse`)
- 10 seed freelancers, fully searchable
- Filter by category, sort by rating/rate
- Save/unsave freelancers

### Freelancer Profile (`/freelancer/:id`)
- Tabbed view: About / Skills / Reviews
- Save to favourites
- Send Message + Hire CTA
- Similar talent sidebar

### Find Work (`/jobs`)
- 8 seed jobs + user-posted jobs
- Filter by category, type (Fixed/Hourly)
- Save/unsave jobs

### Job Detail (`/job/:id`)
- Full description, skills, metadata
- Proposal submission with validation
- Save job button
- Similar jobs sidebar
- Live wallet balance card

### Post a Job (`/post-job`)
- Title, category, type, description, skills, budget
- Full client-side validation
- Adds to global job board instantly

### Wallet (`/wallet`)
- Available balance, escrow, earned, total spent
- **Add funds** — preset or custom amount
- **Withdraw** — validates against balance
- **Escrow hold** — funded on proposal acceptance
- **Release escrow** — one-click release to freelancer
- Transaction history — filterable by type
- All state persists via localStorage

### Dashboard (`/dashboard`)
- Role-aware (Client vs Freelancer view)
- Overview KPIs: balance, escrow, contracts, proposals
- My Jobs tab (client) / Applied Jobs tab (freelancer)
- Proposals tab with Accept & Fund Escrow action
- Activity tab: full transaction log

### Contracts (`/contracts`)
- Active / Completed contract cards
- Milestone tracking — click to toggle done/undone
- Progress bar auto-calculates from milestones
- Release escrow inline with confirmation panel
- Message freelancer shortcut

### Messages (`/messages`, `/messages/:otherId`)
- Full inbox sidebar with conversation list
- Real-time-simulated message thread
- Date dividers, sent/received bubbles
- New conversation — browse freelancers to start chat
- All messages persisted

### Profile & Settings (`/profile`)
- Profile info: name, email, location, bio
- Skills selector (24 options)
- Hourly rate (freelancer mode)
- Security tab (UI demo)
- Switch role (Client ↔ Freelancer)
- Reset demo data (clears localStorage)

### Notifications
- Bell icon in navbar with unread count badge
- Notifications auto-generated by key actions
- Mark all read, clear individual

    Footer.jsx
